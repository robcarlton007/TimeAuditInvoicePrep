## The Four Service Types

Every hour logged at Carlton Edwards falls into one of four service categories. These are separate scopes, often separate contracts, and must be billed to the correct project and phase.

| Service | Entity | What it is | Key signal |
|---|---|---|---|
| **Interior Architecture** | Carlton Architecture PA / Carlton Edwards PLLC | Interior specs, selections, and their application to the architecture — finishes, materials, fixtures, layouts | Project has **no CEC prefix**; note describes selections, finishes, specs, interior details |
| **Architecture** | Carlton Architecture PA / Carlton Edwards PLLC | Structural, framework, and building — generic definition of spaces and generic finishes | Project has **no CEC prefix**; note describes building systems, structural, envelope, generic layouts |
| **CEC (Construction Services)** | Carlton Edwards Construction | Construction management, procurement, contractor coordination, owner's representative services | Project has **CEC prefix**; note describes procurement, logistics, payments, contractor work, scheduling |
| **Furnishings** | Carlton Architecture PA / Carlton Edwards PLLC | Furniture selection, procurement, and installation — FF&E as a standalone scope | Separate furniture contract or phase; note describes furniture products, not architectural finishes |

### Interior Architecture vs. Architecture — The Distinction

> *Example: (ARCH) Generic tile hatch vs. (IA) specific tile layout and placing jollys*

- **Architecture** defines the space generically — dimensions, structure, systems, generic finishes
- **Interior Architecture** defines the specifics within that space — which tile, which finish, which fixture, which material, and exactly how it's applied

When both scopes exist on the same project, an employee's note determines which category the work belongs to. Generic drawings = Architecture. Specific product selection, finish specification, or material coordination = Interior Architecture.

## Workflow Phases Within Each Service Type

### Interior Architecture Workflow

These are the sequential phases of interior architecture work. Each phase has specific activities that belong exclusively to it.

#### Correspondence
Anything associated with the items below — emails, calls, coordination related to interior architecture scope.

#### 01 Selection
- Identification of products
- Identification of finishes
- Identifying product costs
- Selecting a product
- Analyzing the pros/cons of selections (including price)
- Verify availability of the products and finishes

#### 02 Client Management
- Collecting material samples
- Presentation boards
- Client meetings, client approvals

#### 03 Documentation
- Drawings (creating, revising, and updating)
- Schedules (creating, revising, and updating)
- Specification of products and descriptions
- Project manual management

#### 04 Interior Procurement
- Selecting a vendor (seller of product)
- Verification that order matches the design intent
- Verification that the order matches the desired finish
- Verification that the order includes the correct products
- Purchase (initial payment)
- Handoff (anything to get it to the next person)

#### 05 Interior Administration
- Construction admin associated with interiors
- Review of finish samples
- Review of shop drawings from vendor/provider for design intent
- Review of technical drawings
- Task lists management and updates
- Project management meetings — representing interiors
- Warranty binder preparation

## Product / Material Responsibility Matrix

This is the critical split: **who is responsible for what product or material?** The same physical item may be touched by Interior Architecture (spec & finish) and CEC (procurement & installation). Getting the project code wrong here is the most common billing error.

### Interior Architecture Owns (Spec & Finish)

| Material | Interior Architecture Scope |
|---|---|
| Cabinetry | Door style, details, finish, shop drawing review |
| Countertops | Material and finish, shop drawing review |
| Hardware, Cabinetry | Decorative |
| Tile | Material, grout, accessories, layouts |
| Wood Flooring | **Spec and finishes only** |
| Wood Paneling | **Spec and finishes only** |
| Wood Ceilings | **Spec and finishes only** |
| Plumbing fixtures | Except hosebib |
| Lighting fixtures | Integrated and decorative |
| Door Hardware | Excludes pocket door tracks |
| Wall Coverings / Wall Paper | Product and the installation |
| Arch Grille | Style and type, finish (not ordering), shop drawing review, submittal |
| Interior Doors | Finish and design |
| Paint | Product, color, texture, locations, finish, schedule |
| Plaster | Product, color, texture, locations |
| Appliances | Specs, selections, and integration |
| Window Treatment | Decorative fabrics and design |
| Low Voltage | Equipment approval (submittals review) |

### CEC Owns (Procurement, Installation, Structure)

| Material | CEC Scope |
|---|---|
| Concrete | All |
| Masonry | All |
| Steel | All |
| Wood (structural) | All |
| Decking | All |
| Siding | All |
| Soffits | All |
| Pocket door tracks | All (excluded from Interior Arch) |
| Wood Flooring | **Everything but finishes and specs** |
| Wood Paneling | **Everything but finishes and specs** |
| Wood Ceilings | **Everything but finishes and specs** |
| Arch Grille | Stock, standard sizes & sizes, procurement and purchasing for both custom and standard |
| Ditra Heat | All |
| Attic Access Panels | All |
| Plaster | Procurement (not spec/finish) |
| Appliances | Procurement (not spec/selection) |
| Window Treatment | Integrated (not decorative) |

### Split Items — Highest Confusion Risk

These materials are **split** between Interior Architecture and CEC. Both entities touch them, but the *nature of the work* determines which project code to use.

| Material | Interior Architecture | CEC |
|---|---|---|
| **Cabinetry** | Door style, details, finish, shop drawing review | Generically draw, dimension; procurement, installation labor |
| **Countertops** | Material and finish, shop drawing review | Generically draw, dimension; procurement, installation labor |
| **Wood Flooring** | Spec and finishes only | Everything but finishes and specs |
| **Wood Paneling** | Spec and finishes only | Everything but finishes and specs |
| **Wood Ceilings** | Spec and finishes only | Everything but finishes and specs |
| **Plaster** | Product, color, texture, locations | Procurement, everything but finishes and specs |
| **Appliances** | Specs, selections, and integration | Procurement |
| **Arch Grille** | Style/type, finish, shop drawing review | Stock sizes, procurement, purchasing |
| **Interior Doors** | Finish and design | Finish only (CEC on casework projects) |
| **Window Treatment** | Decorative fabrics and design | Integrated |

## Classification Decision Rules

When Luca evaluates a timesheet entry, apply these rules in order:

### Rule 1: Check the project name
- **CEC prefix** → CEC (Construction Services). The note must describe CEC-type activities. If it describes design work, flag as potential miscoding.
- **No CEC prefix** → Architecture or Interior Architecture. Proceed to Rule 2.

### Rule 2: What does the note describe?
- **Structural, framework, building envelope, generic space definition** → Architecture
- **Specific materials, finishes, fixtures, selections, interior details** → Interior Architecture
- **Furniture products, FF&E, furniture procurement/installation** → Furnishings
- **Contractor coordination, procurement, logistics, payments, scheduling** → CEC

### Rule 3: Check the material/product
If the note references a specific material, check the responsibility matrix above:
- Is this a **spec/finish** activity? → Interior Architecture
- Is this a **procurement/installation/structural** activity? → CEC
- Is this a **split item**? → The verb determines the owner:
  - *Selected, specified, reviewed finish, chose color* → Interior Architecture
  - *Ordered, received, installed, tracked shipment, paid vendor* → CEC

### Rule 4: Check the workflow phase
Map the described activity to the workflow phases above. If the activity belongs to a different workflow than the one billed, flag it:
- Selection/documentation/client management activities on a CEC project → flag
- Procurement/shipping/payment activities on an Interior Architecture project → flag
- Installation activities on an Interior Architecture project (should be Furnishings or CEC) → flag

### Rule 5: Ambiguous entries
Some activities legitimately span categories:
- "Client meeting" can occur in any service type — the project code determines the billing
- "Review" can occur in Architecture, Interior Architecture, or CEC depending on what is being reviewed
- "Coordination" depends on who is being coordinated with — consultants (Arch), vendors (IA/CEC), contractors (CEC)

When the note is too vague to classify, flag for review rather than guessing.

## Common Confusion Patterns

### 1. Interior Architecture work coded to CEC (or vice versa)
**Most common error.** Employee selects tile finishes (Interior Architecture) but codes to the CEC project because the client is the same. The CEC prefix is the definitive indicator — if the project says CEC, only CEC activities belong there.

### 2. CEC procurement coded to Interior Architecture
Employee orders materials or coordinates with a vendor (CEC procurement activity) but codes it to the architecture project because they were selecting the product earlier in the same week.

### 3. Furnishings coded to Interior Architecture
Furniture procurement and installation are separate from interior architecture finish work. If the note describes furniture products specifically (not built-in architectural elements), it belongs in Furnishings.

### 4. Interior Administration confused with Construction Administration
"05 Int. Administration" (Interior Architecture) covers review of finish samples, shop drawings for design intent, and technical drawing review — all from the interior architecture perspective. "CA" (Construction Administration) is the architect's oversight role during construction. They overlap but bill differently.

### 5. Split materials billed to wrong entity
Wood flooring specs and finishes = Interior Architecture. Wood flooring installation and everything else = CEC. The same physical material, two different project codes depending on whether you're specifying it or installing it.
